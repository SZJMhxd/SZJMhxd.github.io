    switch多选择结构

    • 多选择结构还有一个实现方式就是switch case语句。
    • switch case语句判断一个变量与一系列值中某个值是否相等,每个值称为一个分支。
    • switch语句中的变量类型可以是:
      • byte, short, int 或者char
      • 从JavaSE7开始switch支持字符串String类型了
      • 同时case标签必须为字符串常量或字面量
