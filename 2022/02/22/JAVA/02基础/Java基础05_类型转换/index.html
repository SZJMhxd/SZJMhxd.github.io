类型转换

类型转换

  • 由于Java是强类型语言，所以要进行有些运算的时候的，需要用到类型转换。
  • 低————————————————————> 高
    byte, short, char -> int -> long -> float -> double
  • 小数优先级高
  • 运算中，不同类型的数据先转化为同一类型，然后进行运算。

强制类型转换

（类型）变量名 高–低

自动类型转换

低–高

注意

  1. 不能对布尔值进行转换
  2. 不能把对象类型转为不想干的类型
  3. 高–低 需要强转 低–高自动
  4. 转换的时候可能存在内存溢出或者精度问题

当操作数比较大时，注意溢出问题

  1. JDK7新特性，数字之间可以用下划线分割

  2. 数据大，溢出
