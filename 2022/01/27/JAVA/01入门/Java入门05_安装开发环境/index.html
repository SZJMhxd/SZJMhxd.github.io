卸载JDK

  1. 我的电脑 –> 右键 –> 属性 –> 高级系统设置 –> 环境变量 –> JAVA_HOME

  2. 找到变量值所指示的文件夹 –> 删除整个文件夹

  3. 清理环境变量 –> JAVA_HOME

  4. 清理Path –> 清理与JAVA_HOME或JAVA相关的环境变量 确定

  5. 确定删除 –> 打开命令提示行 –> 输入java -version

  6. 卸载成功
