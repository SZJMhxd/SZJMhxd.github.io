使用IDEA开发

使用记事本编写代码非常麻烦，工程量很大，调试不方便，编译完成后在提示行才能看见错误

什么是IDE 集成开发环境_百度百科 (baidu.com)

安装IDEA

  1. 最好使用专业版

  2. 可以更改安装位置

    1. 勾选 64位 和 Java

创建一个Java项目并运行

  1. 点击New Project

  2. 新建Java项目_1

    1. 项目类型
    2. SDK版本 如果为红色从下拉箭头中选择已经下载的JDK或者从IDEA安装新的JDK
    3. 如果没有下一步提示按钮 先选择左侧 Empyt Project新建个空项目 再回来新建Java项目

  3. 新建Java项目_2

    填写项目名称 选择项目地址

  4. 新建Java项目_3

  5. 输入psvm回车 快速生成主方法

  6. 输入sout回车 快速输出语句

  7. 点击箭头 运行程序

  8. 控制台输出了Hello,World!

  9. 如果出现错误 比如这里我们少输入一个分号

    错误的地方会有红色波浪线提示 右侧进度条也有提示

    强制运行的话下方控制台也会有详细提示

设置IDEA

设置字体大小

  1. 依次点击 File –> Settings

    2.搜索font –> 点击搜索到的选项

  2. 设置 字体 字号 行间距

其他优化
