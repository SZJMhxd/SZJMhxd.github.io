可变参数

  • JDK 1.5开始，Java支持传递同类型的可变参数给一个方法。

  • 在方法声明中，在指定参数类型后加一个省略号(..)。

  • 一个方法中只能指定一个可变参数,它必须是方法的最后一个参数。任何普通的参数必须在它
    之前声明。

  • 可变参数必须是一个类型
