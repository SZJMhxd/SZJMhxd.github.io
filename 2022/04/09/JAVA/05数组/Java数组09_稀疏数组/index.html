稀疏数组

需求:编写五子棋游戏中，有存盘退出和续上盘的功能

因为该二维数组的很多值是默认值0，因此记录了很多没有意义的数据。

  • 当一个数组中大部分元素为0,或者为同一值的数组时，可以使用稀疏数组来保存该数组
  • 稀疏数组的处理方式是:
    • 记录数组一共有几行几列，有多少个不同值
    • 把具有不同值的元素和行列及值记录在一个小规模的数组中，从而缩小程序的规模
