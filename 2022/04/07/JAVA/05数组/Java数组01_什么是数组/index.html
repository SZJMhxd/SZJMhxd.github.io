数组的定义

  • 数组是相同类型数据的有序集合
  • 数组描述的是相同类型的若干个数据,按照一-定的先后次序排列组合而成
  • 其中,每一个数据称作一个数组元素,每个数组元素可以通过一 个下标来访问它们.
