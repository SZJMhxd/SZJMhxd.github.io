简单小结类与对象

  1. 类与对象
    • 类是一个模板，抽象；对象是一个具体的实例
  2. 方法
    • 定义、调用
  3. 对应的引用
    • 引用类型：基本类型（8个）除了基本类型都可以叫做引用类型
    • 对象通过引用操作（地址） 栈–>堆
  4. 属性 字段field 成员变量
    • 默认初始化：
      • 数字：0 0.0
      • char：u0000
      • boolean：false
      • 其他：null
    • 修饰符：属性类型 属性名= 属性值
  5. 对象的创建和使用
    • 必须使用new 关键字创造对象，构造器 Person SZJM = new Person();
    • 对象的属性 SZJM.name
    • 对象的方法 SZJM.touchFish()
  6. 类：
    • 静态的属性 属性
    • 动态的行为 方法
  7. 三大特征
    • 封装
    • 继承
    • 多态
